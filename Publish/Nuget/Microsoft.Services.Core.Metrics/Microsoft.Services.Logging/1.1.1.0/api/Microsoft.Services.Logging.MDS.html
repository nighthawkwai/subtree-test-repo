GenerateMdsConfigFragments

Generates MDS config fragments for all the events that a given assembly could emit using this logging library. It searches for interfaces in the assembly that are attributed with the logging attributes defined in this library, validates the interfaces and generates MDS config fragments for them NOTE: This is only a partial MDS config. It generates only 'Events' and 'EventStreamingAnnotations' sections of the MDS config, with some of the settings left blank
